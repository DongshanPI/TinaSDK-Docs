## 1.2 适用范围

主要适用于以下平台：
• Allwinner 软件平台Tina v3.0 版本及以上。
• Allwinner 硬件平台R 系列（R6，R11，R16，R18，R30，R40，R328，R331, R329,R818, R818B, R528…）。
• Allwinner 硬件平台MR 系列（MR133, MR813, MR813B…）。
• Allwinner 硬件平台H 系列（H133…）。
• Allwinner 硬件平台V 系列（V853…）。
