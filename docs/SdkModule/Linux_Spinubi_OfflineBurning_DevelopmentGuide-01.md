## 2 名词解释

| 词   | 义                   |
| ---- | -------------------- |
| UBI  | unsorted block image |
| PEB  | physical erase block |
| LEB  | logical erase block  |

PEB 和logical block 关系

```
1 PEB = 1 logical block
1 logical block = 2 physical blocks
```
