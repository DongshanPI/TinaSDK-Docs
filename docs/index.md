# 东山Pi简述(Introduction)

## 什么是东山Pi系列开发板？
**注意: PI系列主板由于没有任何利润空间，所以只能提供在线入门文档和相关的参考资料，没有专门的技术支持和服务，请大家根据自己情况进行合理购买。**<br>

东山Pi系列开发是深圳百问网公司为想学习嵌入式Linux开发同学专门设计定做的低价超高性价比开发板,我们的在线文档更倾向于入门 ，对于深入部分我们会根据SOC特性设计专门对应的底板出相关的深入课程。
